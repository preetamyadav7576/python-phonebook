# 📱 Python Phone Book Management System

A simple **Phone Book Management System** built using **Python** and **Object-Oriented Programming (OOP)** concepts. This project allows users to store, validate, search, and display contact information through a command-line interface.

---

## 🚀 Features

* ➕ Add new contacts
* 📋 Display all saved contacts
* 🔍 Search contacts by name
* 📞 Validate phone numbers
* 🏗️ Built using Python OOP concepts
* 💻 Simple and beginner-friendly CLI application

---

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

---

## 📚 OOP Concepts Used

* Classes and Objects
* Constructor (`__init__`)
* Instance Methods
* Class Variables
* Class Methods (`@classmethod`)
* Static Methods (`@staticmethod`)
* Encapsulation of contact data

## 🔍 Phone Number Validation

The program checks that:

* The phone number contains only digits.
* The phone number has at least 8 digits.
